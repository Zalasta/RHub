🛡 Protection des données
Tes données RHub (rappels, notes, feedbacks) sont stockées dans Safari et survivent aux mises à jour de l'app. Sauvegarde régulièrement.
📅
Importer calendrier .ics
Apple Calendar · Outlook
›
Comment exporter depuis Apple Calendar
  1. Mac : Calendrier → Fichier → Exporter → Calendrier…
  2. Choisir le calendrier → Enregistrer (.ics)
  3. Transférer le fichier ici (AirDrop, iCloud Drive)
Comment exporter depuis Outlook
  1. Outlook Web → Calendrier → ⚙️ → Paramètres
  2. Calendrier → Événements → Exporter
  3. Télécharger le fichier .ics

Importer notes .txt / .md
Apple Notes exportées en texte
›
Comment exporter une note Apple
  1. Ouvrir la note sur iPhone
  2. Appuyer sur Partager → Enregistrer dans Fichiers (.txt)
  3. Répéter pour chaque note
  4. Importer ici (sélection multiple possible)
